À propos de Wdes SAS

Wdes SAS est une entreprise Française qui aime concevoir des outils libres et open-source. L'entreprise offre également des services payants à d'autres entreprises ou particuliers.

Code LTS

Code LTS est un projet de l'entreprise Wdes qui a pour but de maintenir et moderniser du vieux code pour prolonger son support.
Vous pouvez découvrir Code-LTS sur GitHub.

Sudo-Bot

Le projet Sudo-Bot est orienté vers la conception d'images Docker, les outils de déploiement et proposer des actions GitHub pour les développeurs.
Vous pouvez découvrir Sudo-Bot sur GitHub.

Air-Balloon

Ce projet apportera un panneau de contrôle aux mainteneurs de projets open-source pour les accompagner dans leurs tâches du quotidien.
Vous pouvez découvrir Air-Balloon sur GitHub.

Wdes

L'espace Wdes est l'endroit où nos outils et projets vivent sur GitHub.
Vous y trouverez principalement des librairies PHP et des outils NodeJS/Typescript, ainsi que du code Rust.

Services
